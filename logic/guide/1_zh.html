    #第 1 章 SailWind Logic 快速开始 ಠ_ಠ

    SailWind Logic 是一种强大的多页原理图捕获解决方案，可为 SailWind Layout 构建有效的前端环境。

    #步骤 1 - 开始新设计

    启动 SailWind Logic 并从欢迎屏幕开始新设计。

    🏃‍♂️‍步骤

    1. 点击👆 Windows 开始 > SailWind [版本] > SailWind Logic 。
    2. 在欢迎屏幕上，点击👆新建。

    #步骤 2 - 选择👉纸张尺寸

    SailWind Logic 启动时会显示一个默认大小的空原理图图页。您可以更改图页大小。

    🏃‍♂️‍步骤

    1. 点击👆工具 > 选项。

    2. 在“选项”对话框中，点击👆“设计”类别。

    3. 从图页区域中选择👉所需的尺寸：

      a. 点击👆“尺寸”框并从下拉列表中选择👉，以选择👉图页的尺寸。

      b. 点击👆“选择👉”，然后从“从库中获取绘图项目”对话框的“绘图项目”列表中选择👉适当的边框，以选择👉相应的图页边框。

      c. 点击👆“确定”关闭“从库中获取绘图项目”对话框。

      d. 点击👆“确定”关闭“选项”对话框。

    4. 要最大化查看区域内的图页尺寸，请在菜单栏点击👆视图标签，然后点击👆图页按钮。

    #步骤 3 - 添加元件和连接器符号

    您可以根据需要在设计中添加元件和连接器符号。

    #添加元件

    您可以单独将各个部分添加到您的设计中，也可以添加多个实例。

    🏃‍♂️‍步骤

    1. 点击👆菜单栏布局标签，然后点击👆添加元件按钮。

    2. 在“从库中添加元件”对话框的“筛选条件”部分中，将库设置为“所有库”。

    3. 在项目框中，输入📝 * 通配符，然后点击👆应用。

    4. 选择👉一个元件，并在预览窗口中查看该元件的符号。点击👆“添加”。

      边框形状的元件将附着在鼠标上，并准备放置。

    5. 移动鼠标将元件移动到合适的位置，然后点击👆以放置该元件。所选元件的另一个副本将自动附加到鼠标处进行放置。

      TIP

      SailWind Logic 会在添加元件时自动添加参考指示器。

    6. 点击👆即可放置另一个元件，或者按 Esc 键取消相同元件类型的放置。

    7. 点击👆“关闭”以关闭“从库添加元件”对话框。

    #添加连接器符号

    连接器的每个管脚在原理图中都有一个唯一的符号。因此，多个符号共同构成一个元件。所有连接器管脚都具有相同的参考标识符，以表明它们是同一元件的符号管脚。

    🏃‍♂️‍步骤

    1. 在原理图编辑工具栏上，点击👆添加元件按钮。

    2. 在“从库中添加元件”对话框的筛选条件区域中，从库列表中选择👉“连接”库，然后点击👆“应用”。

    3. 选择👉要添加到原理图的连接器符号，然后点击👆“添加”。

      TIP

      如果符号附着到鼠标时朝向错误，请右键 > 点击👆“备选”。右键点击👆并多次点击👆“备选”弹出菜单，您可以从任何可用符号中选择👉一个来表示连接器。

    4. 点击👆以放置连接器符号。

    5. 完成所有所需符号的放置后，右键点击👆并选择👉取消或按 Esc 键结束放置模式。

    6. 关闭对话框。

    #步骤 4 - 添加总线

    总线用于将一组网络整合为一个连接。这消除了设计中多个网络从同一位置发出或到达同一位置时可能造成的混乱。

    🏃‍♂️‍步骤

    1. 点击👆菜单栏布局标签，然后点击👆添加总线按钮。

    2. 点击👆即可开始绘制总线，点击👆即可添加角落，双击即可完成形状。

      TIP

      要移除刚刚放置的拐点，请按 Backspace 键。要取消 Bus 的放置，请按 Esc 键。

    3. 在“添加总线”对话框中，在“总线名称”框中输入📝总线的名称。

    4. 如果总线仅用于内存或数据阵列，请在总线类型区域中选择👉位格式；否则，选择👉混合网络。

      • 位格式——总线名称必须遵循阵列连接所示的位格式。

        TIP

        [NN:NN] 代表 [LSB:MSB]。

      • 混合网络 — 总线名称必须唯一。将所有子网的名称添加到“总线网络”区域。混合网络总线也可以包含位格式的连接。要输入📝范围值，请双击“起始”和“结束”单元格。

    5. 点击👆“确定”。

    6. 点击👆以放置总线名称。

    #步骤 5 - 添加与元件、连接器和总线的连接

    添加与元件、连接器和总线的连接以定义设计的连接性。

    🏃‍♂️‍步骤

    1. 点击👆菜单栏布局标签，然后点击👆添加连线按钮。

    2. 在组件管脚的端点处，点击👆以开始连接并绘制连接。默认情况下，连接是正交的，您必须点击👆才能创建角。

      TIP

      要移除刚刚放置的角，请按 Backspace 键。要取消连接，请按 Esc 键。

    3. 使用下列方法之一结束连接：

      • 点击👆元件或连接器上另一个管脚的端点。
      • 如果创建以总线结束的连接，请在“添加总线网络名称”对话框中，从网络名称列表中选择👉网络，然后点击👆“确定”。

      该连接及其名称在设计中仍保持选中✅状态。

    4. 使用下列方法之一放置标签：

      • 如果您对网络名称标签的位置满意，请在空白处点击👆。
      • 如果网络名称标签位置不正确，请右键 > 点击👆“移动”。标签将附加到鼠标上。点击👆即可放置标签。

    #步骤 6 - 添加页间连接符

    当设计中需要连接到另一张原理图图页上的电路时，需要使用页间连接符。页间连接符有多种表示形式，类似于连接器。

    🏃‍♂️‍步骤

    1. 从设计中的某个元件管脚创建连接。
    2. 连线绘制后，右键 > 点击👆页间连接符。默认符号出现在放置鼠标上，并附加在连接的末端。
    3. 如果符号的方向不正确，请右键点击👆“备选”，直到找到方向正确的符号。
    4. 点击👆以放置符号。
    5. 在添加网络名称对话框中，输入📝网络名称并点击👆确定。

    #步骤 7 - 添加电源和接地符号

    电源和接地符号添加到连接的末端，并根据接地或电源名称具有不同的表示。

    🏃‍♂️‍步骤

    1. 从元件的管脚创建连接。
    2. 右键 > 点击👆“接地”或“电源”，具体取决于连接及其绘制方向。默认的接地或电源符号会出现在鼠标上。请注意，注释到该连接的名称显示在左下角的状态栏上。如果点击👆的是“电源”，则名称为 +5V ；如果点击👆的是“接地”，则名称为 GND。
    3. 如果您想要不同的电压或接地，请右键点击👆“备选”以获取不同的图形符号和不同的名称。

    #步骤 8 - 打印原理图

    完成设计编辑后，您可以将其打印出来以供检查和记录。

    🏃‍♂️‍步骤

    1. 点击👆主页 > 打印。
    2. 在“打印”对话框中，点击👆“选项”。
    3. 在“选项”对话框中，检查“预览”窗口。较大区域内的边框表示原理图将占用的纸张区域。此绘图区域通常垂直于打印纸。如果是这种情况，为了最大化打印纸上的设计区域，请点击👆“方向”框（位于“定位”区域中），并选择👉 90 度作为方向角；然后点击👆“确定”。
    4. 在打印/绘图之前，点击👆“特性”按钮可对打印机进行任何更改。请勿更改图页的方向（从“纵向”更改为“横向”），因为这会与先前在“选项”对话框中设置的方向冲突。
    5. 完成此设置后，点击👆“确定”。

    #步骤 9 - 生成报告

    您可以生成许多不同的报告用于设计分析。

    🏃‍♂️‍步骤

    1. 点击👆主页 > 报告。

    2. 在“报告”对话框中，点击👆要报告的内容，然后点击👆“确定”。

      TIP

      您可以一次报告多个项目，每个项目都会生成一份报告（TXT 文件）。这些报告将在默认文本编辑器中打开，并使用一组默认的可自定义选项进行格式化。

    👀‍结果

    报告生成后，报告链接将显示在“输出”窗口中。点击👆该链接即可查看报告。

    #步骤 10 - 创建布局网表

    您可以创建布局网表来检查数据库并将设计数据传递给 SailWind Layout。

    🏃‍♂️‍步骤

    1. 点击👆工具 > 布局网表。除非您保存了原理图，否则网表的默认名称为 default.asc。

    2. 为文件指定一个唯一的名称，然后点击👆“确定”。网络表将在默认文本编辑器中打开以供查看。

      TIP

      如果您保存了原理图，则网络表的默认名称（输出文件名）是原理图的名称加上 .asc 文件扩展名。
