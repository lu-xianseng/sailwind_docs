    #з¬¬ 11 з«  еӣҫйЎө

    жӮЁеҸҜд»ҘдҪҝз”ЁеӣҫйЎөеҜ№и®ҫи®ЎиҝӣиЎҢйҖ»иҫ‘еҲҶеҢәгҖӮжӮЁиҝҳеҸҜд»ҘдёәеӣҫйЎөж·»еҠ иҮӘе®ҡд№үиҫ№жЎҶе’Ңеӯ—ж®өгҖӮ

    #зј–иҫ‘еӣҫйЎө

    дҪҝз”ЁвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶеҸҜеңЁе·ҘдҪңеҢәдёӯзј–иҫ‘еҪ“еүҚеҺҹзҗҶеӣҫзҡ„еӣҫйЎөйӣҶгҖӮдҪҝз”ЁвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶпјҢжӮЁеҸҜд»ҘеңЁеӣҫйЎөйӣҶдёӯж·»еҠ е’ҢеҲ йҷӨеӣҫйЎөпјҢд»ҘеҸҠдҝ®ж”№еӣҫйЎөйӣҶзҡ„еҗҚз§°е’Ңзј–еҸ·йЎәеәҸгҖӮжңҖеӨҡеҸҜд»ҘеҲӣе»ә 1024 еј еӣҫйЎөгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. зӮ№еҮ»рҹ‘Ҷи®ҫзҪ® > еӣҫйЎөгҖӮ

    2. зӮ№еҮ»рҹ‘ҶвҖңе·Ізј–еҸ·зҡ„еӣҫйЎөвҖқеҲ—иЎЁжЎҶдёӯзҡ„жҹҗдёӘеӣҫйЎөпјҢз„¶еҗҺзӮ№еҮ»рҹ‘ҶвҖңжҹҘзңӢвҖқд»ҘеңЁе·ҘдҪңеҢәдёӯжҹҘзңӢиҜҘеӣҫйЎөгҖӮ

    3. зӮ№еҮ»рҹ‘ҶвҖңж·»еҠ вҖқе°Ҷж–°еӣҫйЎөж·»еҠ еҲ°еҲ—иЎЁдёӯгҖӮж–°еӣҫйЎөе°ҶжҳҫзӨәеңЁе·ҘдҪңеҢәдёӯгҖӮ

    4. зӮ№еҮ»рҹ‘ҶвҖңе·Ізј–еҸ·зҡ„еӣҫйЎөвҖқеҲ—иЎЁжЎҶдёӯзҡ„ж–°еӣҫйЎөпјҢз„¶еҗҺзӮ№еҮ»рҹ‘ҶвҖңйҮҚе‘ҪеҗҚвҖқд»Ҙе°Ҷе…¶йҮҚе‘ҪеҗҚгҖӮ

      TIP

      з©әж јдёҚжҳҜеӣҫйЎөеҗҚз§°дёӯзҡ„жңүж•Ҳеӯ—з¬ҰгҖӮ

    5. еҰӮжһңйңҖиҰҒпјҢиҜ·д»ҺвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶдёӯжү§иЎҢд»ҘдёӢд»»ж„Ҹйҷ„еҠ ж“ҚдҪңпјҡ

      • зӮ№еҮ»рҹ‘ҶвҖңеҲ йҷӨвҖқеҸҜд»ҺеҲ—иЎЁдёӯз§»йҷӨеӣҫйЎөгҖӮ
      • зӮ№еҮ»рҹ‘ҶвҖңеҗ‘дёҠвҖқеҸҜжӣҙж”№еӣҫйЎөзҡ„зј–еҸ·е’ҢдҪҚзҪ®гҖӮеӣҫйЎөеҖјжӣҙж”№дёә n-1гҖӮ
      • зӮ№еҮ»рҹ‘ҶвҖңеҗ‘дёӢвҖқеҸҜжӣҙж”№еӣҫйЎөзҡ„зј–еҸ·е’ҢдҪҚзҪ®гҖӮеӣҫйЎөеҖјжӣҙж”№дёә n $+1$ гҖӮ

    #еҲӣе»әиҮӘе®ҡд№үеӣҫйЎөиҫ№жЎҶ

    жӮЁеҸҜд»ҘдҪҝз”Ёз§°дёәвҖңеӯ—ж®өвҖқзҡ„дё“з”Ёж–Үжң¬еӯ—з¬ҰдёІеҲӣе»әеёҰжңүж Үйўҳж Ҹзҡ„иҮӘе®ҡд№үеӣҫйЎөиҫ№жЎҶгҖӮе°Ҷеӯ—ж®өж·»еҠ еҲ°еӣҫйЎөиҫ№жЎҶзҡ„ж Үйўҳж Ҹж—¶пјҢиҝҷдәӣеӯ—ж®өдјҡиҮӘеҠЁдј йҖ’еҲ°жҜҸдёӘж–°еӣҫйЎөгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. з»ҳеҲ¶ 2D зәҝиҫ№жЎҶгҖӮпјҲиҜ·еҸӮйҳ…еҲӣе»ә 2D зәҝйЎ№зӣ®гҖӮпјү
    2. еҗ‘ 2D зәҝеҜ№иұЎж·»еҠ еӯ—ж®өгҖӮпјҲиҜ·еҸӮйҳ…ж·»еҠ еӯ—ж®өгҖӮпјү
    3. е°ҶеҜ№иұЎдҝқеӯҳеҲ°зәҝжқЎеә“гҖӮпјҲиҜ·еҸӮйҳ…е°Ҷз»ҳеӣҫйЎ№зӣ®ж·»еҠ еҲ°еә“дёӯгҖӮпјү

    #ж·»еҠ еӯ—ж®ө

    еӯ—ж®өз”ЁдҪңж–ҮжЎЈдёӯеҸҜиғҪеҸ‘з”ҹеҸҳеҢ–зҡ„ж•°жҚ®зҡ„еҚ дҪҚз¬ҰгҖӮжӮЁеҸҜд»Ҙе°Ҷеӯ—ж®өж·»еҠ еҲ°еҺҹзҗҶеӣҫжҲ–д»»дҪ• 2D зәҝеҜ№иұЎгҖӮеҰӮжһңжӮЁйҖүжӢ©рҹ‘үдәҶ 2D зәҝеҜ№иұЎпјҢеҲҷж·»еҠ зҡ„еӯ—ж®өдјҡеңЁж”ҫзҪ®ж—¶иҮӘеҠЁдёҺе…¶еҗҲе№¶гҖӮ

    TIP

    еӯ—ж®өжңүдёӨз§Қзұ»еһӢпјҡзі»з»ҹеӯ—ж®өе’Ңз”ЁжҲ·еӯ—ж®өгҖӮ

    жӮЁж— жі•дҝ®ж”№зі»з»ҹеӯ—ж®өпјӣе…¶еҖјжқҘиҮӘзі»з»ҹгҖӮ

    з”ЁжҲ·еӯ—ж®өжҳҜжӮЁеҲӣе»әзҡ„иҮӘе®ҡд№үеӯ—ж®өгҖӮз”ЁжҲ·еӯ—ж®өзҡ„еҗҚз§°жҲ–еҖјеӯ—з¬ҰдёІжІЎжңүйҷҗеҲ¶гҖӮз”ЁжҲ·еӯ—ж®өд»…еңЁеҲӣе»әе®ғд»¬зҡ„еҺҹзҗҶеӣҫдёӯеҸҜз”ЁгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. йҖүжӢ©рҹ‘үд»»дҪ•еҶ…е®№жҲ– 2D зәҝеҜ№иұЎпјҢеҸій”® > еұһжҖ§ > ж·»еҠ гҖӮ

    2. еңЁеҗҚз§°еҲ—иЎЁдёӯпјҢйҖүжӢ©рҹ‘үдёҖдёӘзі»з»ҹеӯ—ж®өжҲ–иҫ“е…Ҙрҹ“қиҮӘе®ҡд№үеӯ—ж®өеҗҚз§°гҖӮ

    3. еңЁвҖңеҖјвҖқжЎҶдёӯпјҢиҫ“е…Ҙрҹ“қиҰҒжҳҫзӨәзҡ„еҖјгҖӮиҜ·жіЁж„ҸпјҢзі»з»ҹеӯ—ж®өж— жі•дҪҝз”ЁвҖңеҖјвҖқжЎҶпјҢеӣ дёәиҜҘеҖјжәҗиҮӘжӮЁзҡ„зі»з»ҹгҖӮ

    4. иҰҒе°Ҷеӯ—ж®өж”ҫзҪ®еңЁзІҫзЎ®зҡ„ X, Y еқҗж ҮдҪҚзҪ®пјҢиҜ·еңЁ X е’Ң Y жЎҶдёӯиҫ“е…Ҙрҹ“қеҖјгҖӮ

      TIP

      еҰӮжһңзӮ№еҮ»рҹ‘ҶвҖңзЎ®е®ҡвҖқж—¶жӯӨеӯ—ж®өдёәз©әзҷҪпјҢеҲҷиҜҘеӯ—ж®өе°Ҷйҷ„еҠ еҲ°йј ж ҮпјҢзӣҙеҲ°жӮЁзӮ№еҮ»рҹ‘Ҷд»ҘжҢҮзӨәдҪҚзҪ®гҖӮ

    5. еңЁж—ӢиҪ¬еҲ—иЎЁдёӯпјҢйҖүжӢ©рҹ‘үжүҖйңҖзҡ„ж—ӢиҪ¬еәҰж•°гҖӮ

      TIP

      е°–з«Ҝж—ӢиҪ¬еҸҜд»ҘжҳҜ 0 еәҰжҲ– 90 еәҰгҖӮ

    6. еңЁвҖңеӨ§е°ҸвҖқжЎҶдёӯпјҢиҫ“е…Ҙрҹ“қжүҖйңҖзҡ„еӯ—дҪ“еӨ§е°ҸгҖӮ

      з¬”еҲ’еӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 10 еҲ° 1000 жҜ«зұід№Ӣй—ҙпјӣзі»з»ҹеӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 1 еҲ° 72 зӮ№д№Ӣй—ҙгҖӮ

    7. еҜ№дәҺз¬”еҲ’еӯ—дҪ“пјҢеңЁвҖңзәҝе®ҪвҖқжЎҶдёӯиҫ“е…Ҙрҹ“қзәҝе®ҪгҖӮ

    8. еҜ№дәҺзі»з»ҹеӯ—дҪ“пјҢеңЁеӯ—дҪ“еҲ—иЎЁдёӯйҖүжӢ©рҹ‘үиҰҒдҪҝз”Ёзҡ„еӯ—дҪ“гҖӮ

      жӮЁиҝҳеҸҜд»ҘзӮ№еҮ»рҹ‘ҶиҰҒеә”з”Ёзҡ„зі»з»ҹеӯ—дҪ“ж ·ејҸпјҡB иЎЁзӨәзІ—дҪ“гҖҒI иЎЁзӨәж–ңдҪ“гҖҒU иЎЁзӨәдёӢеҲ’зәҝгҖӮ

    9. иҰҒи®ҫзҪ®еҜ№йҪҗж–№ејҸпјҢиҜ·зӮ№еҮ»рҹ‘ҶжүҖйңҖзҡ„ж°ҙе№іе’ҢеһӮзӣҙйҖүйЎ№гҖӮ

    10. зӮ№еҮ»рҹ‘ҶвҖңзЎ®е®ҡвҖқгҖӮ

    #е°Ҷж–Үжң¬еӯ—з¬ҰдёІжӣҙж”№дёәеӯ—ж®ө

    жӮЁеҸҜд»ҘдҪҝз”ЁвҖңе»әз«Ӣеӯ—ж®өвҖқе‘Ҫд»Өе°ҶзҺ°жңүж–Үжң¬еӯ—з¬ҰдёІжӣҙж”№дёәеӯ—ж®өгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. йҖүжӢ©рҹ‘үдёҖдёӘж–Үжң¬еӯ—з¬ҰдёІпјҢеҸій”® > зӮ№еҮ»рҹ‘ҶвҖңе»әз«Ӣеӯ—ж®өвҖқгҖӮ

    2. еңЁеҗҚз§°еҲ—иЎЁдёӯпјҢиҫ“е…Ҙрҹ“қеӯ—ж®өеҗҚз§°жҲ–йҖүжӢ©рҹ‘үзі»з»ҹеӯ—ж®өгҖӮ

    3. иҰҒе°Ҷеӯ—ж®өж”ҫзҪ®еңЁзІҫзЎ®зҡ„ X,Y еқҗж ҮдҪҚзҪ®пјҢиҜ·еңЁ X е’Ң Y жЎҶдёӯиҫ“е…Ҙрҹ“қеҖјгҖӮ

      TIP

      еҰӮжһңзӮ№еҮ»рҹ‘ҶвҖңзЎ®е®ҡвҖқж—¶жӯӨеӯ—ж®өдёәз©әзҷҪпјҢеҲҷиҜҘеӯ—ж®өе°Ҷйҷ„еҠ еҲ°йј ж ҮпјҢзӣҙеҲ°жӮЁзӮ№еҮ»рҹ‘Ҷд»ҘжҢҮзӨәдҪҚзҪ®гҖӮ

    4. еңЁж—ӢиҪ¬еҲ—иЎЁдёӯпјҢйҖүжӢ©рҹ‘үжүҖйңҖзҡ„ж—ӢиҪ¬еәҰж•°гҖӮ

      ж—ӢиҪ¬еҸҜд»ҘжҳҜ 0 еәҰжҲ– 90 еәҰгҖӮ

    5. еңЁвҖңе°әеҜёвҖқжЎҶдёӯпјҢиҫ“е…Ҙрҹ“қжүҖйңҖзҡ„еӯ—дҪ“еӨ§е°ҸгҖӮ

      з¬”еҲ’еӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 10 еҲ° 1000 жҜ«зұід№Ӣй—ҙпјӣзі»з»ҹеӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 1 еҲ° 72 зӮ№д№Ӣй—ҙгҖӮ

    6. еҜ№дәҺз¬”еҲ’еӯ—дҪ“пјҢеңЁвҖңзәҝе®ҪвҖқжЎҶдёӯиҫ“е…Ҙрҹ“қзәҝе®ҪгҖӮ

    7. еҜ№дәҺзі»з»ҹеӯ—дҪ“пјҢеңЁеӯ—дҪ“еҲ—иЎЁдёӯйҖүжӢ©рҹ‘үиҰҒдҪҝз”Ёзҡ„еӯ—дҪ“гҖӮ

      жӮЁиҝҳеҸҜд»ҘзӮ№еҮ»рҹ‘ҶиҰҒеә”з”Ёзҡ„зі»з»ҹеӯ—дҪ“ж ·ејҸпјҡB иЎЁзӨәзІ—дҪ“гҖҒI иЎЁзӨәж–ңдҪ“гҖҒU иЎЁзӨәдёӢеҲ’зәҝгҖӮ

    8. иҰҒи®ҫзҪ®еҜ№йҪҗж–№ејҸпјҢиҜ·зӮ№еҮ»рҹ‘ҶжүҖйңҖзҡ„ж°ҙе№іе’ҢеһӮзӣҙйҖүйЎ№гҖӮ

    9. зӮ№еҮ»рҹ‘ҶвҖңзЎ®е®ҡвҖқгҖӮ

    #з®ЎзҗҶеӯ—ж®ө

    дҪҝз”ЁвҖңеӯ—ж®өвҖқеҜ№иҜқжЎҶз®ЎзҗҶеӨҡдёӘеӯ—ж®өгҖӮжӮЁеҸҜд»Ҙз®ЎзҗҶж•ҙдёӘеҺҹзҗҶеӣҫжҲ– 2D зәҝеҜ№иұЎдёӯзҡ„еӯ—ж®өгҖӮ

    #з®ЎзҗҶзӨәж„Ҹеӣҫдёӯзҡ„жүҖжңүеӯ—ж®ө

    еҰӮжһңжӮЁжңӘйҖүжӢ©рҹ‘үд»»дҪ•еҶ…е®№пјҢиҪҜд»¶е°ҶеҲ—еҮәеҺҹзҗҶеӣҫдёӯдҪҝз”Ёзҡ„жүҖжңүеӯ—ж®өгҖӮзі»з»ҹеӯ—ж®өжҳҫзӨәеңЁзі»з»ҹеҢәеҹҹдёӯпјҢжӮЁе®ҡд№үзҡ„еӯ—ж®өжҳҫзӨәеңЁз”ЁжҲ·еҢәеҹҹдёӯгҖӮжӮЁеҸӘиғҪз®ЎзҗҶз”ЁжҲ·еӯ—ж®өгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. йҖүжӢ©рҹ‘үдёҚеҢ…еҗ«д»»дҪ•еҶ…е®№жҲ– 2D зәҝеҜ№иұЎпјҢеҸій”®зӮ№еҮ»рҹ‘ҶвҖңеӯ—ж®ө...вҖқгҖӮ
    2. еңЁвҖңз”ЁжҲ·вҖқеҢәеҹҹпјҢзӮ№еҮ»рҹ‘ҶвҖңж·»еҠ вҖқгҖӮ
    3. еңЁеҗҚз§°еҲ—дёӯиҫ“е…Ҙрҹ“қеҗҚз§°гҖӮ
    4. еңЁеҖјеҲ—дёӯиҫ“е…Ҙрҹ“қдёҖдёӘеҖјгҖӮ
    5. иҰҒд»ҺеҺҹзҗҶеӣҫдёӯеҲ йҷӨеӯ—ж®өпјҢиҜ·йҖүжӢ©рҹ‘үеӯ—ж®өеҗҚз§°пјҢз„¶еҗҺзӮ№еҮ»рҹ‘ҶвҖңеҲ йҷӨвҖқгҖӮйҷҗеҲ¶пјҡжӮЁж— жі•д»ҺеҺҹзҗҶеӣҫдёӯеҲ йҷӨзі»з»ҹеӯ—ж®өгҖӮ
    6. иҰҒзј–иҫ‘еӯ—ж®өзҡ„еҗҚз§°жҲ–еҖјпјҢиҜ·йҖүжӢ©рҹ‘үиҰҒзј–иҫ‘зҡ„еҗҚз§°жҲ–еҖјпјҢз„¶еҗҺзӮ№еҮ»рҹ‘Ҷзј–иҫ‘гҖӮ

    #з®ЎзҗҶ 2D зәҝйЎ№зӣ®дёӯзҡ„еӯ—ж®ө

    еҰӮжһңжӮЁйҖүжӢ©рҹ‘үдәҶ 2D зәҝеҜ№иұЎпјҢиҪҜд»¶дјҡеҲ—еҮәдёҺиҜҘеҜ№иұЎе…іиҒ”зҡ„жүҖжңүеӯ—ж®өгҖӮзі»з»ҹеӯ—ж®өжҳҫзӨәеңЁзі»з»ҹеҢәеҹҹдёӯпјҢжӮЁе®ҡд№үзҡ„еӯ—ж®өжҳҫзӨәеңЁз”ЁжҲ·еҢәеҹҹдёӯгҖӮжӮЁеҸҜд»Ҙд»Һ 2D зәҝдёӯеҲ йҷӨзі»з»ҹеӯ—ж®өпјҢдҪҶе®ғд»¬еңЁеҺҹзҗҶеӣҫдёӯд»Қз„¶еҸҜз”ЁгҖӮ

    рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

    1. йҖүжӢ©рҹ‘үдёҖдёӘ 2D зәҝйЎ№зӣ®пјҢз„¶еҗҺеҸій”®зӮ№еҮ»рҹ‘ҶвҖңеӯ—ж®өвҖқгҖӮиҝҷе°Ҷжү“ејҖвҖңеӯ—ж®өвҖқеҜ№иҜқжЎҶгҖӮ

    2. иҰҒд»ҺдәҢз»ҙиЎҢйЎ№зӣ®дёӯеҲ йҷӨзі»з»ҹеӯ—ж®өпјҢиҜ·еңЁвҖңзі»з»ҹвҖқеҢәеҹҹдёӯйҖүжӢ©рҹ‘үеӯ—ж®өеҗҚз§°пјҢз„¶еҗҺзӮ№еҮ»рҹ‘ҶвҖңеҲ йҷӨвҖқгҖӮиҜҘзі»з»ҹеӯ—ж®өеңЁеҺҹзҗҶеӣҫдёӯд»Қз„¶еҸҜз”ЁгҖӮ

    3. иҰҒд»ҺдәҢз»ҙиЎҢйЎ№зӣ®дёӯеҲ йҷӨз”ЁжҲ·еӯ—ж®өпјҢиҜ·еңЁвҖңз”ЁжҲ·вҖқеҢәеҹҹдёӯйҖүжӢ©рҹ‘үеӯ—ж®өеҗҚз§°пјҢз„¶еҗҺзӮ№еҮ»рҹ‘ҶвҖңеҲ йҷӨвҖқгҖӮиҜҘеӯ—ж®өеңЁзӨәж„Ҹеӣҫдёӯд»Қз„¶еҸҜз”ЁгҖӮ

    4. иҰҒзј–иҫ‘з”ЁжҲ·еӯ—ж®өзҡ„еҖјпјҢиҜ·йҖүжӢ©рҹ‘үиҰҒзј–иҫ‘зҡ„еҖјпјҢз„¶еҗҺзӮ№еҮ»рҹ‘Ҷзј–иҫ‘гҖӮ

      TIP

      зј–иҫ‘з”ЁжҲ·еӯ—ж®өж—¶пјҢиҜ·жіЁж„Ҹд»ҘдёӢдәӢйЎ№пјҡ

      • жӮЁж— жі•зј–иҫ‘з”ЁжҲ·еӯ—ж®өзҡ„еҗҚз§°гҖӮ
      • жӮЁж— жі•ж·»еҠ еӯ—ж®өд»Ҙдҫӣд»ҘеҗҺеңЁ 2D зәҝдёӯдҪҝз”ЁгҖӮ
