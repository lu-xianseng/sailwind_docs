зј–иҫ‘еӣҫйЎө#

дҪҝз”ЁвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶеҸҜеңЁе·ҘдҪңеҢәдёӯзј–иҫ‘еҪ“еүҚеҺҹзҗҶеӣҫзҡ„еӣҫйЎөйӣҶгҖӮдҪҝз”ЁвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶпјҢжӮЁеҸҜд»ҘеңЁеӣҫйЎөйӣҶдёӯж·»еҠ е’ҢеҲ йҷӨеӣҫйЎөпјҢд»ҘеҸҠдҝ®ж”№еӣҫйЎөйӣҶзҡ„еҗҚз§°е’Ңзј–еҸ·йЎәеәҸгҖӮжңҖеӨҡеҸҜд»ҘеҲӣе»ә 1024 еј еӣҫйЎөгҖӮ

рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

  1. зӮ№еҮ»вҖңи®ҫзҪ® > еӣҫйЎөгҖӮ

  2. зӮ№еҮ»вҖңзј–еҸ·еӣҫйЎөвҖқеҲ—иЎЁжЎҶдёӯзҡ„жҹҗдёӘеӣҫйЎөпјҢз„¶еҗҺзӮ№еҮ»вҖңжҹҘзңӢвҖқд»ҘеңЁе·ҘдҪңеҢәдёӯжҹҘзңӢиҜҘеӣҫйЎөгҖӮ

  3. зӮ№еҮ»вҖңж·»еҠ вҖқе°Ҷж–°еӣҫйЎөж·»еҠ еҲ°еҲ—иЎЁдёӯгҖӮж–°еӣҫйЎөе°ҶжҳҫзӨәеңЁе·ҘдҪңеҢәдёӯгҖӮ

  4. зӮ№еҮ»вҖңзј–еҸ·еӣҫйЎөвҖқеҲ—иЎЁжЎҶдёӯзҡ„ж–°еӣҫйЎөпјҢз„¶еҗҺзӮ№еҮ»вҖңйҮҚе‘ҪеҗҚвҖқд»Ҙе°Ҷе…¶йҮҚе‘ҪеҗҚгҖӮ

    TIP

    з©әж јдёҚжҳҜеӣҫйЎөеҗҚз§°дёӯзҡ„жңүж•Ҳеӯ—з¬ҰгҖӮ

  5. еҰӮжһңйңҖиҰҒпјҢиҜ·д»ҺвҖңеӣҫйЎөвҖқеҜ№иҜқжЎҶдёӯжү§иЎҢд»ҘдёӢд»»ж„Ҹйҷ„еҠ ж“ҚдҪңпјҡ

    • зӮ№еҮ»вҖңеҲ йҷӨвҖқеҸҜд»ҺеҲ—иЎЁдёӯз§»йҷӨеӣҫйЎөгҖӮ
    • зӮ№еҮ»вҖңеҗ‘дёҠвҖқеҸҜжӣҙж”№еӣҫйЎөзҡ„зј–еҸ·е’ҢдҪҚзҪ®гҖӮеӣҫйЎөеҖјжӣҙж”№дёә n-1гҖӮ
    • зӮ№еҮ»вҖңеҗ‘дёӢвҖқеҸҜжӣҙж”№еӣҫйЎөзҡ„зј–еҸ·е’ҢдҪҚзҪ®гҖӮеӣҫйЎөеҖјжӣҙж”№дёә n $+1$ гҖӮ

еҲӣе»әиҮӘе®ҡд№үеӣҫйЎөиҫ№жЎҶ#

жӮЁеҸҜд»ҘдҪҝз”Ёз§°дёәвҖңеӯ—ж®өвҖқзҡ„дё“з”Ёж–Үжң¬еӯ—з¬ҰдёІеҲӣе»әеёҰжңүж Үйўҳж Ҹзҡ„иҮӘе®ҡд№үеӣҫйЎөиҫ№жЎҶгҖӮе°Ҷеӯ—ж®өж·»еҠ еҲ°еӣҫйЎөиҫ№жЎҶзҡ„ж Үйўҳж Ҹж—¶пјҢиҝҷдәӣеӯ—ж®өдјҡиҮӘеҠЁдј йҖ’еҲ°жҜҸдёӘж–°еӣҫйЎөгҖӮ

рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

  1. з»ҳеҲ¶ 2D зәҝиҫ№жЎҶгҖӮпјҲиҜ·еҸӮйҳ…еҲӣе»ә 2D зәҝйЎ№зӣ®гҖӮпјү
  2. еҗ‘ 2D зәҝеҜ№иұЎж·»еҠ еӯ—ж®өгҖӮпјҲиҜ·еҸӮйҳ…ж·»еҠ еӯ—ж®өгҖӮпјү
  3. е°ҶеҜ№иұЎдҝқеӯҳеҲ°зәҝжқЎеә“гҖӮпјҲиҜ·еҸӮйҳ…е°Ҷз»ҳеӣҫйЎ№зӣ®ж·»еҠ еҲ°еә“гҖӮпјү

ж·»еҠ еӯ—ж®ө#

еӯ—ж®өз”ЁдҪңж–ҮжЎЈдёӯеҸҜиғҪеҸ‘з”ҹеҸҳеҢ–зҡ„ж•°жҚ®зҡ„еҚ дҪҚз¬ҰгҖӮжӮЁеҸҜд»Ҙе°Ҷеӯ—ж®өж·»еҠ еҲ°еҺҹзҗҶеӣҫжҲ–д»»дҪ• 2D зәҝеҜ№иұЎгҖӮеҰӮжһңжӮЁйҖүжӢ©дәҶ 2D зәҝеҜ№иұЎпјҢеҲҷж·»еҠ зҡ„еӯ—ж®өдјҡеңЁж”ҫзҪ®ж—¶иҮӘеҠЁдёҺе…¶еҗҲе№¶гҖӮ

TIP

еӯ—ж®өжңүдёӨз§Қзұ»еһӢпјҡзі»з»ҹеӯ—ж®өе’Ңз”ЁжҲ·еӯ—ж®өгҖӮ

жӮЁж— жі•дҝ®ж”№зі»з»ҹеӯ—ж®өпјӣе…¶еҖјжқҘиҮӘзі»з»ҹгҖӮ

з”ЁжҲ·еӯ—ж®өжҳҜжӮЁеҲӣе»әзҡ„иҮӘе®ҡд№үеӯ—ж®өгҖӮз”ЁжҲ·еӯ—ж®өзҡ„еҗҚз§°жҲ–еҖјеӯ—з¬ҰдёІжІЎжңүйҷҗеҲ¶гҖӮз”ЁжҲ·еӯ—ж®өд»…еңЁеҲӣе»әе®ғд»¬зҡ„еҺҹзҗҶеӣҫдёӯеҸҜз”ЁгҖӮ

рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

  1. йҖүжӢ©д»»дҪ•еҶ…е®№жҲ– 2D зәҝеҜ№иұЎпјҢеҸій”® > еұһжҖ§ > ж·»еҠ гҖӮ

  2. еңЁеҗҚз§°еҲ—иЎЁдёӯпјҢйҖүжӢ©дёҖдёӘзі»з»ҹеӯ—ж®өжҲ–й”®е…ҘиҮӘе®ҡд№үеӯ—ж®өеҗҚз§°гҖӮ

  3. еңЁвҖңеҖјвҖқжЎҶдёӯпјҢй”®е…ҘиҰҒжҳҫзӨәзҡ„еҖјгҖӮиҜ·жіЁж„ҸпјҢзі»з»ҹеӯ—ж®өж— жі•дҪҝз”ЁвҖңеҖјвҖқжЎҶпјҢеӣ дёәиҜҘеҖјжәҗиҮӘжӮЁзҡ„зі»з»ҹгҖӮ

  4. иҰҒе°Ҷеӯ—ж®өж”ҫзҪ®еңЁзІҫзЎ®зҡ„ X,Y еқҗж ҮдҪҚзҪ®пјҢиҜ·еңЁ X е’Ң Y жЎҶдёӯй”®е…ҘеҖјгҖӮ

    TIP

    еҰӮжһңзӮ№еҮ»вҖңзЎ®е®ҡвҖқж—¶жӯӨеӯ—ж®өдёәз©әзҷҪпјҢеҲҷиҜҘеӯ—ж®өе°Ҷйҷ„еҠ еҲ°йј ж ҮпјҢзӣҙеҲ°жӮЁзӮ№еҮ»д»ҘжҢҮзӨәдҪҚзҪ®гҖӮ

  5. еңЁж—ӢиҪ¬еҲ—иЎЁдёӯпјҢйҖүжӢ©жүҖйңҖзҡ„ж—ӢиҪ¬еәҰж•°гҖӮ

    TIP

    е°–з«Ҝж—ӢиҪ¬еҸҜд»ҘжҳҜ 0 еәҰжҲ– 90 еәҰгҖӮ

  6. еңЁвҖңеӨ§е°ҸвҖқжЎҶдёӯпјҢй”®е…ҘжүҖйңҖзҡ„еӯ—дҪ“еӨ§е°ҸгҖӮ

    з¬”еҲ’еӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 10 еҲ° 1000 жҜ«зұід№Ӣй—ҙпјӣзі»з»ҹеӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 1 еҲ° 72 зӮ№д№Ӣй—ҙгҖӮ

  7. еҜ№дәҺз¬”еҲ’еӯ—дҪ“пјҢеңЁвҖңзәҝе®ҪвҖқжЎҶдёӯй”®е…Ҙзәҝе®ҪгҖӮ

  8. еҜ№дәҺзі»з»ҹеӯ—дҪ“пјҢеңЁеӯ—дҪ“еҲ—иЎЁдёӯйҖүжӢ©иҰҒдҪҝз”Ёзҡ„еӯ—дҪ“гҖӮ

    жӮЁиҝҳеҸҜд»ҘзӮ№еҮ»иҰҒеә”з”Ёзҡ„зі»з»ҹеӯ—дҪ“ж ·ејҸпјҡB иЎЁзӨәзІ—дҪ“гҖҒI иЎЁзӨәж–ңдҪ“гҖҒU иЎЁзӨәдёӢеҲ’зәҝгҖӮ

  9. иҰҒи®ҫзҪ®еҜ№йҪҗж–№ејҸпјҢиҜ·зӮ№еҮ»жүҖйңҖзҡ„ж°ҙе№іе’ҢеһӮзӣҙйҖүйЎ№гҖӮ

  10. зӮ№еҮ»вҖңзЎ®е®ҡвҖқгҖӮ

е°Ҷж–Үжң¬еӯ—з¬ҰдёІжӣҙж”№дёәеӯ—ж®ө#

жӮЁеҸҜд»ҘдҪҝз”ЁвҖңе»әз«Ӣеӯ—ж®өвҖқе‘Ҫд»Өе°ҶзҺ°жңүж–Үжң¬еӯ—з¬ҰдёІжӣҙж”№дёәеӯ—ж®өгҖӮ

рҹҸғвҖҚвҷӮпёҸвҖҚжӯҘйӘӨ

  1. йҖүжӢ©дёҖдёӘж–Үжң¬еӯ—з¬ҰдёІпјҢеҸій”®зӮ№еҮ»пјҢз„¶еҗҺзӮ№еҮ»вҖңе»әз«Ӣеӯ—ж®өвҖқгҖӮ

  2. еңЁеҗҚз§°еҲ—иЎЁдёӯпјҢй”®е…Ҙеӯ—ж®өеҗҚз§°жҲ–йҖүжӢ©зі»з»ҹеӯ—ж®өгҖӮ

  3. иҰҒе°Ҷеӯ—ж®өж”ҫзҪ®еңЁзІҫзЎ®зҡ„ X,Y еқҗж ҮдҪҚзҪ®пјҢиҜ·еңЁ X е’Ң Y жЎҶдёӯй”®е…ҘеҖјгҖӮ

    TIP

    еҰӮжһңзӮ№еҮ»вҖңзЎ®е®ҡвҖқж—¶жӯӨеӯ—ж®өдёәз©әзҷҪпјҢеҲҷиҜҘеӯ—ж®өе°Ҷйҷ„еҠ еҲ°йј ж ҮпјҢзӣҙеҲ°жӮЁзӮ№еҮ»д»ҘжҢҮзӨәдҪҚзҪ®гҖӮ

  4. еңЁж—ӢиҪ¬еҲ—иЎЁдёӯпјҢйҖүжӢ©жүҖйңҖзҡ„ж—ӢиҪ¬еәҰж•°гҖӮ

    ж—ӢиҪ¬еҸҜд»ҘжҳҜ 0 еәҰжҲ– 90 еәҰгҖӮ

  5. еңЁвҖңеӨ§е°ҸвҖқжЎҶдёӯпјҢй”®е…ҘжүҖйңҖзҡ„еӯ—дҪ“еӨ§е°ҸгҖӮ

    з¬”еҲ’еӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 10 еҲ° 1000 жҜ«зұід№Ӣй—ҙпјӣзі»з»ҹеӯ—дҪ“еӨ§е°Ҹеҝ…йЎ»д»ӢдәҺ 1 еҲ° 72 зӮ№д№Ӣй—ҙгҖӮ

  6. еҜ№дәҺз¬”еҲ’еӯ—дҪ“пјҢеңЁвҖңзәҝе®ҪвҖқжЎҶдёӯй”®е…Ҙзәҝе®ҪгҖӮ

  7. еҜ№дәҺзі»з»ҹеӯ—дҪ“пјҢеңЁеӯ—дҪ“еҲ—иЎЁдёӯйҖүжӢ©иҰҒдҪҝз”Ёзҡ„еӯ—дҪ“гҖӮ

    жӮЁиҝҳеҸҜд»ҘзӮ№еҮ»иҰҒеә”з”Ёзҡ„зі»з»ҹеӯ—дҪ“ж ·ејҸпјҡB иЎЁзӨәзІ—дҪ“гҖҒI иЎЁзӨәж–ңдҪ“гҖҒU иЎЁзӨәдёӢеҲ’зәҝгҖӮ

  8. иҰҒи®ҫзҪ®еҜ№йҪҗж–№ејҸпјҢиҜ·зӮ№еҮ»жүҖйңҖзҡ„ж°ҙе№іе’ҢеһӮзӣҙйҖүйЎ№гҖӮ

  9. зӮ№еҮ»вҖңзЎ®е®ҡвҖқгҖӮ
