    #HiSpeed Rules 对话框

    💃访问方式：设置>设计规则 > 选择👉层次级别 > 高速按钮

    使用此对话框设置高速规则。

    WARNING

    检查高速规则需要 Electrodynamic Checking (EDC)许可选项。

    区域描述
    平行布线• 平行布线规则 (Parallelism) — 指定第1845页所述的平行布线长度与间距值。在条件规则设置对话框中指定的平行布线规则，其优先级高于本对话框中的规则设置。
    • 串联布线规则 (Tandem) — 指定第1864页所述的串联布线长度与间距值。在条件规则设置对话框中指定的串联布线规则，其优先级高于本对话框中的规则设置。
    • 干扰源设置 (Aggressor) — 指定该对象集合中的元素将作为第1808页所述的信号干扰源。
    屏蔽• 屏蔽复选框 (Shield checkbox) — 指定自动布线连接到平面层的走线，以为选定网络提供电磁干扰屏蔽。
    例外情况：当使用过孔进行屏蔽时，屏蔽规则不适用。
    限制条件：
    • 如果设计中没有平面层，则屏蔽区域中的选项不可用。
    • SailWind Layout 不会自动布线屏蔽网络，也不会检查屏蔽规则。
    • 间距 (Gap) — 指定屏蔽走线与被屏蔽网络之间的净空距离。
    • 使用网络列表 (Use net list) — 列出与要实施屏蔽走线的平面层关联的所有网络。
    规则区域
    长度指定允许的最小和最大长度。包括每个连接管脚离散长度值的一半（如果组件已分配离散长度）。
    残桩长度指定从 T 型连接点到走线末端的最大距离，见。
    延迟（纳秒）指定允许的最小和最大延迟。基线延迟计算使用走线参数、在 Layer Parameter Setup 对话框中设置的板材值，并需要至少一个相邻平面层。计算中最多使用两个相邻平面层。如果平面层是分割/混合平面层，则必须在层上有平面形状。平面区域的形状和边框的位置和边框在计算中都被忽略。
    延迟值可在 Pin Pair Properties和 Net Properties对话框中查看。
    限制：• 此规则仅在运行 Tools > Verify Design后验证，在设置 High Speed 检查检查 Delay 时。不由在线 DRC验证。
    电容（皮法）指定允许的最小和最大电容。基线电容计算使用走线参数、在 Layer Parameter Setup 对话框中设置的板材值，并需要至少一个相邻平面层。计算中最多使用两个相邻平面层。如果平面层是分割/混合平面层，则必须在层上有平面形状。平面区域的形状和边框的位置和边框在计算中都被忽略。
    电容值可在 Net Properties对话框中查看。
    限制：• 不能为管脚对设置电容。• 此规则仅在运行 Tools > Verify Design后验证，在设置 High Speed 检查检查 Capacitance 时。不由在线 DRC验证。
    阻抗（欧姆）指定允许的最小和最大阻抗。基线阻抗计算使用走线参数、在 Layer Parameter Setup 对话框中设置的板材值，并需要至少一个相邻平面层。计算中最多使用两个相邻平面层。如果平面层是分割/混合平面层，则必须在层上有平面形状。平面区域的形状和边框的位置和边框在计算中都被忽略。
    阻抗值可在 Pin Pair Properties和 Net Properties对话框中查看。
    限制：此规则仅在运行 Tools > Verify Design后验证，在设置 High Speed 检查检查 Impedance 时。不由在线 DRC验证。
    匹配某些布线器可以自动布线长度匹配的走线。例如，差分对的走线具有匹配的长度以避免信号之间的时序偏差。
    • Match lengths 复选框 — 指定自动布线长度匹配的走线。• Tolerance — 指定匹配长度组中最短和最长的走线之间允许的最大差异。
    限制：SailWind Layout 不检查长度匹配规则。
    删除按钮删除当前规则层次结构级别的非默认高速规则。
    限制：不能删除默认高速规则。
    TIP

    在 Conditional Rule Setup 对话框中指定的并行和串联规则会覆盖此对话框中指定的规则。

    💃访问方式：Electrical Net Rules 对话框 > 高速按钮

    为电气网络分配最小/最大长度规则并创建匹配长度组。

    TIP

    检查高速规则需要 Electrodynamic Checking (EDC)许可选项。

    对象

    字段描述
    长度为在"Electrical Net Rules 对话框"中选择👉的电气网络设置最小和最大长度。包括每个连接管脚离散长度值的一半（如果组件已分配离散长度）。
    匹配长度选中✅此复选框将使在"Electrical Net Rules 对话框"中选择👉的电气网络成为匹配长度组的成员。清除复选框将从现有匹配长度组中移除选定的电气网络。
    nets from an existing matched length group.
    公差设置长度匹配的容差值。

    #HYP Export 对话框

    💃访问方式：选择👉主页 > 导出，从"保存类型"下拉列表中选择👉"Hyp Files (*.hyp)"；然后点击👆保存

    使用 HYP Export 对话框以 HYP 文件形式导出设计。

    描述

    有关 HYP 文件格式和创建"BoardSim 友好"设计的信息，请参阅 PADS HyperLynx 在线帮助。SailWind Layout 将 Value、Tolerance、Voltage、PADS HyperLynx 和 PowerGround 属性传递给 HYP 文件。BoardSim 使用这些属性获取电阻和电容的值，并传输固定电压网络的信息。

    WARNING

    只有 PADS HyperLynx v8.0 或更高版本可以打开此对话框导出的文件。创建的.hyp文件是 v2.34 格式文件。

    字段描述
    输出区域
    网络和网络类导出网络和网络类的属性。
    电源和地网络导出电源和地网络的属性。
    .REF IC 自动映射文件创建一个 PADS HyperLynx .ref 文件，该文件将设计中的 IC 参考标志映射到表示 IC 的 BoardSim 模型。BoardSim 使用这些映射在您选择👉网络进行仿真时自动加载 IC 模型。
    限制：仅当组件具有带值的 HyperLynx.Model 属性时，此复选框才可用。
    未布线导出未布线的网络。
    覆铜平面未布线选项区域使用以下选项导出覆铜平面信息：• 导出覆铜平面的剖面线边框 — 导出灌铜和平面区域的剖面线边框。要求：此设置会在.hyp 文件中创建额外结构，是电源完整性分析所必需的。• 导出覆铜平面边框 — 仅导出覆铜平面边框（无剖面线）。
    提示仅在输出区域中选择👉未布线时可用。
    假设层指定实现未布线网络的层。
    超过曼哈顿长度的百分比指定估计布线长度的值。提示此值将曼哈顿长度的一定百分比添加到布线长度中，以考虑间接布线路径。网络长度基于管脚对之间的曼哈顿距离，即 Delta X 加上 Delta Y。

    #IDF Export 对话框

    💃访问方式：主页> 导出 > 选择👉 IDF 文件 > 保存

    使用 IDF Export 对话框在 SailWind Layout 和机械设计系统之间交换设计数据。您可以导出 IDF 文件以将板框、禁止区、组件和孔导出到机械设计系统。

    TIP

    设置任何 IDF 特定的元件高度信息、钻孔信息或元件边框信息以获得更准确的 IDF 导出。

    字段描述
    IDF 文件名称您要导出的文件的名称。将创建.emn（板和放置）和.emp（元件库）文件。
    形状层指定包含设计中要发送到机械设计系统的元件边框信息的层。
    格式指定要使用的 IDF 版本。
    Pro/ENGINEER Wildfire 支持选中✅此复选框可在字符是 Pro/ENGINEER Wildfire 的非法字符时将其转换为下划线(_)。清除此复选框以允许所有字符在 IDF 文件中导出。
    选择👉导出项目区域使用这些复选框选择👉要导出到形状层列表中层的项目。
    点击👆所有项目选择👉此区域中的所有内容：• 板外形 (Board Outline) — 包含板轮廓、挖空区域及安装孔。
    • 板钻孔 (Board Drills) — 与安装孔相关的钻孔信息，包括钻孔直径与镀覆状态。
    • 顶层元件 (Parts-Top) — 安装在板顶层的元器件及其位置信息。
    • 底层元件 (Parts-Bottom) — 安装在板底层的元器件及其位置信息。
    • 元件钻孔 (Part Drills) — 与元件引脚相关的钻孔信息，包括钻孔直径与镀覆状态。此选项在IDF 2.0版本中不可用，且当"顶层元件"和"底层元件"复选框未勾选✅时将被禁用。
    • 过孔钻孔 (Via Drills) — 与过孔相关的钻孔信息，包括钻孔直径与镀覆状态。此选项在IDF 2.0版本及导入过程中不可用。
    • 布局禁布区 (Place Keepouts) — 包含具有高度限制的布局禁布区及其位置信息。IDF文件可包含以下数据：
    • 板级布局禁布区
    • 在"选项"对话框＞"绘图"类别＞"文本与走线"子类别（第1553页）中为整板定义的顶层与底层元件高度限制
    • 布线禁布区 (Routing Keepouts) — 走线禁布区及其位置信息。仅支持顶层、底层、双面、内层（仅限IDF 3.0）及所有层的走线禁布区，不支持单内层上的走线禁布区。
    限制：此选项对于 IDF 2.0 不可用。
    • 过孔禁止区 — 过孔禁止区及其位置。IDF 仅支持适用于所有层的过孔禁止区。导入的过孔禁止区始终设置在所有层上。此选项对于 IDF 2.0 不可用。
    顶部/底部指定要导出的组件的最小高度。
    提示
    小于这些高度的组件不会被导出。如果元件由于最小高度值未被导出，则会将消息写入状态日志文件。
    设计单位指定此文件的设计单位。

    • 过孔钻孔 (Via Drills) — 与过孔关联的钻孔数据，包括钻孔直径及镀覆状态。此选项在IDF 2.0版本及数据导入过程中不可用。
    • 布局禁布区 (Place Keepouts) — 包含带高度限制的布局禁布区及其位置坐标。IDF文件可支持以下数据：
    • 板级放置禁止区• 为整个板定义的顶部和底部组件高度限制。
    在选项对话框 > 文本和线子类别中找到此设置• 布线禁止区 — 走线禁止区及其位置。仅支持顶部、底部、两者、内部（仅 IDF 3.0）和所有层的走线禁止区。不支持单个内层的走线禁止区。
    此选项对于 IDF 2.0 不可用。• 过孔禁止区 — 过孔禁止区及其位置。IDF 仅支持适用于所有层的过孔禁止区。导入的过孔禁止区始终设置在所有层上。此选项对于 IDF 2.0 不可用。

    #JEDEC Array Pinning 对话框

    💃访问方式：工具 > PCB 封装编辑器 > 工具 > 分配 JEDEC 阵列管脚

    使用 JEDEC Array Pinning 对话框按照 JEDEC 标准为阵列中的每个管脚分配字母数字字符串。

    描述

    管脚行从上到下用字母 A 开始编号。不使用字母 I、O、Q、S、X 和 Z。

    焊盘样式指定焊盘的样式：普通焊盘、热焊盘或反焊盘。
    热和反焊盘显示配置控制用于分割/混合层和CAM 负片平面（用于 RS-274X 输出）的热和反焊盘的大小和形状。
    另请参阅："设计规则与焊盘栈 - 热和反焊盘"。
    提示• 从 PADS 9.2 开始，槽孔的热的大小、形状和方向不再从槽孔的长度、钻孔尺寸和方向派生，而是继承自普通焊盘。• 将内径和外径值设置为相同以获得与平面的实心连接（灌铜）。当前焊盘直径用作内径，外径设置为默认同网络焊盘到角规则。对于反焊盘，直径最初设置为遵循当前默认焊盘到铜设计规则。如果在选项对话框铜平面类别 > 热子类别中选择👉使用设计规则进行热和反焊盘，则忽略外径，改用间距规则，除非外径小于内径。然而，内径和外径选项始终控制灌铜。

    参见"创建自定义焊盘形状"了解更多信息。

    方形 — 大小、宽度（如果孔是槽孔）、角类型和半径。如果焊盘样式为热：内大小、外大小、辐条、辐条角度、辐条宽度。
    环形 — 直径和内径
    奇数 — 直径
    钻孔尺寸如果跳线是通孔跳线，则指定钻孔尺寸。
